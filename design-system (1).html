18 — AI Implementation Guide
🎨 Design Language
Primary Color: #6c5ce7 (violet-purple) — used for active sidebar nav, primary buttons, badges, and focus states.
Background: #f0f1f5 app canvas, #ffffff cards and sidebar.
Cards: White background, 1px #e4e6ed border, 20px border-radius, subtle shadow.
Typography: Plus Jakarta Sans for headings/labels, DM Sans for body text.
Spacing: 4px base grid. Use multiples: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64px.
✅ Component Rules
Buttons: border-radius: 10px. Primary = #6c5ce7 fill + purple shadow. Secondary = white + border.
Inputs: 1.5px border, 10px radius, focus = primary color border + soft glow ring.
Badges/Tags: border-radius: 9999px. Colored dot prefix + category name.
Nav items: Active = primary bg + white text + box-shadow. Hover = gray-50 bg.
Stat cards: Large bold number, small badge for delta (green up / red down), muted sub-text.
Icons: 20–24px, 2px stroke, line style (not filled), rounded linecap.
⚡ Interaction Patterns
Card hover: translateY(-2px) + shadow-md.
Button hover: Primary gets darker bg + more shadow + translateY(-1px).
Button active: scale(0.97).
Toggle: cubic-bezier bounce animation on knob slide.
Transitions: 200ms ease for most, 300ms bounce for toggles/drawers.
Input focus: 0 0 0 3px rgba(108,92,231, 0.12) ring.
